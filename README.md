# MERN Stack Web App
![image](https://github.com/user-attachments/assets/fbf42b8c-3616-4114-92fe-f595fe3ae949)
### ⚛️ Tech Stack: React.js, Node.js, Express.js, MongoDB, Chakra UI
***
### Setup .env file
```
MONGO_URI=your_mongo_uri
PORT=5000
```
### Run on local
```
npm run build
```
### Start the app
```
npm run start
```
